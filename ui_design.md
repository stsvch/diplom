# Проектирование интерфейса (черновой набросок)

## Общая структура (Layout)

Интерфейс строится на основе **бокового меню (sidebar)** + **верхняя панель (header)**. Содержимое sidebar меняется в зависимости от роли пользователя. Тёмная тема не предусмотрена.

```
┌──────────────────────────────────────────────────┐
│  Header: логотип, поиск, уведомления, профиль    │
├──────────┬───────────────────────────────────────┤
│          │                                       │
│ Sidebar  │         Основной контент              │
│ (меню    │                                       │
│  по      │                                       │
│  роли)   │                                       │
│          │                                       │
├──────────┴───────────────────────────────────────┤
│  Footer (опционально)                            │
└──────────────────────────────────────────────────┘
```

Всё выносится на отдельные Angular-компоненты.

---

## Гость (незарегистрированный)

Гость видит минимальный набор страниц без sidebar.

| Экран | Описание |
|-------|----------|
| **Landing page** | Главная страница платформы — описание, преимущества, призыв к регистрации |
| **Каталог курсов** | Список курсов с фильтрацией по дисциплинам, поиск |
| **Страница курса (превью)** | Описание курса, программа (модули), преподаватель, цена. Кнопка «Записаться» ведёт на регистрацию |
| **Регистрация** | Форма: имя, email, пароль, выбор роли (студент/преподаватель) |
| **Вход** | Email + пароль, ссылка на восстановление |
| **Восстановление пароля** | Ввод email → письмо со ссылкой |

## Студент

### Sidebar-меню

- Дашборд
- Мои курсы
- Каталог курсов
- Календарь
- Сообщения
- Уведомления
- Подписки и платежи
- Профиль

### Экраны

| Экран | Описание |
|-------|----------|
| **Дашборд** | Обзор: активные курсы, прогресс, ближайшие дедлайны, последние оценки, уведомления |
| **Мои курсы** | Список курсов, на которые записан. Карточки с прогресс-баром. Фильтр: активные / завершённые |
| **Каталог курсов** | Все доступные курсы, фильтрация по дисциплине, цене, рейтингу. Поиск |
| **Страница курса** | Структура: модули → уроки. Прогресс по модулям. Оценки. Чат курса |
| **Урок** | Контент урока: текст, видео, файлы. Вложения для скачивания. Кнопки: «Пройдено» / «Далее» |
| **Тест (прохождение)** | Вопросы, таймер (если есть), навигация по вопросам, кнопка «Завершить». Результат после завершения |
| **Задание** | Описание, критерии, дедлайн. Форма сдачи: текст + загрузка файлов. История попыток, статус, оценка |
| **Журнал оценок** | Оценки по всем курсам. Группировка по курсу → модулю. Итоговые баллы |
| **Занятия** | Доступные слоты преподавателей. Запись на занятие. Мои записи. История |
| **Календарь** | Дедлайны заданий и тестов, запланированные занятия |
| **Сообщения** | Список чатов (курсовые + личные). Окно переписки. Отправка вложений |
| **Уведомления** | Лента уведомлений с фильтрацией по типу |
| **Словарь** | Список слов/терминов по дисциплине. Добавление своих слов. Упражнения на заучивание (карточки) |
| **Редактор кода** | Задача, описание, редактор с подсветкой синтаксиса, кнопка «Запустить», результат |
| **Подписки и платежи** | Текущая подписка, история платежей, покупка курса или подписки |
| **Профиль** | Личные данные, аватар, смена пароля |

## Преподаватель

### Sidebar-меню

- Дашборд
- Мои курсы
- Создать курс
- Проверка работ
- Журнал оценок
- Расписание занятий
- Отчёты
- Сообщения
- Уведомления
- Словарь
- Профиль

### Экраны

| Экран | Описание |
|-------|----------|
| **Дашборд** | Обзор: мои курсы, количество студентов, работы на проверку, ближайшие занятия, статистика |
| **Мои курсы** | Список созданных курсов. Статусы: черновик / опубликован / архив. Кол-во студентов |
| **Создание/редактирование курса** | Основные настройки: название, дисциплина, описание, обложка, цена, тип прохождения, оценивание |
| **Редактор структуры курса** | Управление модулями и уроками (drag & drop для порядка). Добавление/удаление |
| **Редактор урока** | Добавление контента: текстовый редактор (WYSIWYG), загрузка файлов, видео, ссылки |
| **Создание теста** | Добавление вопросов разных типов. Настройки: таймер, попытки, дедлайн, перемешивание |
| **Создание задания** | Описание, критерии, дедлайн, попытки, макс. балл |
| **Проверка работ** | Список работ на проверку (задания + открытые вопросы тестов). Фильтр по курсу. Оценивание, комментарии, возврат на доработку |
| **Журнал оценок** | Таблица: студенты x задания/тесты. Ручное редактирование оценок. Экспорт в PDF/Excel |
| **Расписание занятий** | Создание слотов (дата, время, групповое/индивидуальное). Просмотр записавшихся. Управление статусами |
| **Отчёты** | Успеваемость по курсу, по студенту, по группе. Прогресс прохождения. Графики |
| **Словарь** | Управление словарём дисциплины: добавление/редактирование слов, картинки, аудио |
| **Создание задачи по коду** | Описание, язык, начальный код, тест-кейсы, ожидаемый результат |
| **Сообщения** | Чаты курсов + личные сообщения |
| **Уведомления** | Лента уведомлений |
| **Профиль** | Личные данные, аватар |

## Ключевые компоненты интерфейса (общие, переиспользуемые)

| Компонент | Где используется |
|-----------|-----------------|
| **Карточка курса** | Каталог, дашборд, мои курсы |
| **Прогресс-бар** | Мои курсы, страница курса, дашборд |
| **Таблица с сортировкой и фильтрацией** | Журнал, пользователи, платежи, отчёты |
| **Текстовый редактор (WYSIWYG)** | Редактор урока, задания, комментарии |
| **Загрузчик файлов** | Урок, задание, сдача работы, сообщения |
| **Модальные окна** | Подтверждения, быстрое создание, предупреждения |
| **Система уведомлений (toast)** | Всплывающие сообщения об успехе/ошибке |
| **Календарь** | Дедлайны, расписание |
| **Чат** | Курсовой чат, личные сообщения |
| **Редактор кода** | Задачи по программированию (Monaco Editor / CodeMirror) |

## Заметки

- Это черновой набросок, детали будут уточняться при реализации.
- Все элементы интерфейса выносятся на отдельные Angular-компоненты.
- Интерфейс адаптивный (десктоп, планшет, мобильные устройства).
- Язык интерфейса — русский.
