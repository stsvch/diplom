## Что такое Python

Python — язык программирования общего назначения с простым и понятным синтаксисом.
Создан Гвидо ван Россумом в 1991 году. Используется в веб-разработке, анализе данных,
машинном обучении, автоматизации и многих других областях.

## Установка

1. Скачайте установщик с https://www.python.org/downloads/
2. При установке поставьте галочку **Add Python to PATH**
3. Проверьте в терминале: `python --version`

## Первая программа

```python
print("Hello, world!")
```

Запустите файл `hello.py` командой `python hello.py`.

## Переменные

Переменные создаются при первом присваивании. Тип выводится автоматически.

```python
name = "Алиса"          # строка
age = 25                # целое число
height = 1.72           # число с плавающей точкой
is_student = True       # логический тип

print(f"{name}, {age} лет")
```

## Простой калькулятор

```python
a = int(input("Первое число: "))
b = int(input("Второе число: "))
print(f"Сумма: {a + b}")
print(f"Разность: {a - b}")
print(f"Произведение: {a * b}")
print(f"Частное: {a / b}")
```
