# Урок. Что такое производная

**Курс:** Алгебра и начала анализа
**Модуль:** Введение в математический анализ

## Мотивация

Представьте, что вы едете на машине. Спидометр показывает мгновенную скорость —
то есть, с какой скоростью расстояние растёт **прямо сейчас**. Производная функции —
это «спидометр» для любой величины, которая меняется во времени или от какой-то переменной.

## Определение

Производная функции f(x) в точке x₀ — это предел отношения приращения функции
к приращению аргумента при стремлении приращения к нулю:

> f′(x₀) = lim (Δx → 0) [f(x₀ + Δx) − f(x₀)] / Δx

Геометрически f′(x₀) — это **тангенс угла наклона касательной** к графику функции в точке x₀.

## Основные производные

| Функция | Производная |
|---|---|
| c (константа) | 0 |
| x | 1 |
| xⁿ | n · xⁿ⁻¹ |
| sin x | cos x |
| cos x | −sin x |
| eˣ | eˣ |
| ln x | 1/x |

## Правила дифференцирования

- (f + g)′ = f′ + g′
- (f · g)′ = f′ · g + f · g′
- (f / g)′ = (f′ · g − f · g′) / g²
- (f(g(x)))′ = f′(g(x)) · g′(x) — правило цепочки

## Примеры

**Пример 1.** Найти производную f(x) = 3x² + 5x − 7.
> f′(x) = 6x + 5

**Пример 2.** Найти f′(x), если f(x) = sin(2x).
> По правилу цепочки: f′(x) = cos(2x) · 2 = 2cos(2x)

## Задачи

1. Найдите f′(x) для f(x) = x³ − 4x + 1.
2. Найдите f′(x) для f(x) = (2x + 1)⁵.
3. В какой точке касательная к графику f(x) = x² горизонтальна?

## Ответы

1. 3x² − 4
2. 10(2x + 1)⁴
3. В точке x = 0 (т.к. f′(x) = 2x = 0).
